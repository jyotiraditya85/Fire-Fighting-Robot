## üì± Manual Control (Bluetooth Mode)

The robot can be controlled manually using an Android mobile phone via Bluetooth.

### Requirements

* Android smartphone
* Bluetooth module: **HC-05** (pre-installed in robot)
* App: **Bluetooth RC Controller (Base APK)**

---

### üì• Step 1: Install the App

1. Download and install the **Bluetooth RC Controller** base APK on your Android phone.

---

### üîµ Step 2: Enable Bluetooth and Connect

1. Turn **ON Bluetooth** on your phone.
2. Search for nearby Bluetooth devices.
3. Select the device named **`HC-05`**.
4. Pair the device (default PIN is usually `1234` or `0000`).

---

### üéÆ Step 3: Connect Through the App

1. Open the **Bluetooth RC Controller** app.
2. Open the **Options Menu** in the app.
3. Select **Connect Car**.
4. Choose **HC-05** from the list.
5. On the control screen, the signal indicator will change from **Red to Green**, confirming a successful connection.

---

### ‚öôÔ∏è Step 4: App Settings Configuration

1. Open the **Options Menu** again.
2. Select **Settings**.
3. Configure the following options:

   * **Accelerometer Holding Position**:

     * Select **Face Screen Towards User** (this position stops the car).
   * **Data Stream Frequency**:

     * Set to **On Change / Touch**.
4. Save the settings.

---

### ‚úÖ Manual Control Ready

The robot is now ready to be controlled manually using:

* On-screen buttons, or
* Phone accelerometer (tilt control), depending on app mode.

## ü§ñ Automatic Mode

1. Power ON the robot.
2. Ensure no Bluetooth commands are being sent.
3. The robot will automatically:

   * Detect fire using sensors
   * Navigate towards the fire
   * Activate the fire extinguishing mechanism
